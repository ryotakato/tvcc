

# Tavi C Compiler by Rust



## Environment

* Mac Silicon (M4)
* Docker
* Rust


## Dependencies

* Docker
* Docker Compose
* [cross](https://crates.io/crates/cross)
* [cargo-make](https://crates.io/crates/cargo-make)


## Reference

* [低レイヤを知りたい人のためのCコンパイラ作成入門](https://www.sigbus.info/compilerbook)
